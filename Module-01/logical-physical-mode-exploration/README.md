# Packet Tracer - Logical and Physical Mode Exploration

## Objective

Explore Cisco Packet Tracer's logical and physical workspaces and gain familiarity with network devices, cabling, and device management.

## Tasks Completed

- Explored physical network locations and topology.
- Investigated the branch office wiring closet.
- Connected a PC to a switch using an Ethernet cable.
- Connected a laptop to a router using a USB console cable.
- Installed and powered on a backup router.
- Configured the router hostname through the command line interface.

## Skills Learned

- Cisco Packet Tracer navigation
- Physical vs Logical network representation
- Ethernet connectivity
- USB Console connectivity
- Basic Cisco IOS configuration
- Network device management

## Tools Used

- Cisco Packet Tracer

## Outcome

Successfully completed the lab and gained practical experience with network device installation, connectivity, and basic router configuration.
